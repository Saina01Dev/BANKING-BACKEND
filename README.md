# Banking Backend API

This project is a backend banking system built using Node.js and Express.

## 🚀 Features
- User Authentication (Login / Signup)
- Secure Password Hashing
- Transaction Management
- Fraud Detection Service
- RESTful APIs

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcrypt

## 📁 Project Structure
config/
controllers/
middleware/
models/
routes/
services/
server.js

## ▶️ How to Run the Project

1. Clone the repository
git clone https://github.com/Saina01Dev/BANKING-BACKEND.git

2. Install dependencies
npm install

3. Start the server
node server.js

Server will run on:
http://localhost:5000
